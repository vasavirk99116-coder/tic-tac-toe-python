# Tic Tac Toe Game (Python)

This project is a simple Tic Tac Toe game implemented using Python.
Two players can play the game by entering positions from 1 to 9.

## Features
- Two-player game
- Win and draw detection
- Console-based interface

## Technologies Used
- Python

## How to Run
1. Download the file tic_tac_toe.py
2. Run using:
   python tic_tac_toe.py# tic-tac-toe-python
